## Bundled libraries and tools

### 🗺 WallyMapEditor

A Brawlhalla map editor. Launched from Odd Brawlhalla's **Map Studio** tab.

- **Project:** https://github.com/moffel1020/WallyMapEditor
- **Copyright (c) 2023 moffel1020**
- **License:** MIT

### 🗺 WallyMapSpinzor2

Brawlhalla map (`.xml`) file format library used by WallyMapEditor.

- **Project:** https://github.com/AllHailCheese/WallyMapSpinzor2
- **Copyright (c) 2023 AllHailCheese**
- **License:** MIT

### 🗜 BrawlhallaSwz

Decryption / encryption library for Brawlhalla's `.swz` archive format.

- **Project:** https://github.com/AllHailCheese/BrawlhallaSwz
- **Copyright (c) 2023 AllHailCheese**
- **License:** MIT

### 🎞 SwiffCheese

SWF shape rendering library used to render legend art.

- **Project:** https://github.com/AllHailCheese/SwiffCheese
- **Copyright (c) 2024 AllHailCheese**
- **License:** MIT

### 🔡 AbcDisassembler

ActionScript Bytecode (ABC) disassembler used by the SWZ tooling.

- **Project:** https://github.com/moffel1020/AbcDisassembler
- **Copyright (c) 2024 moffel1020**
- **License:** MIT

### 🦴 WallyAnmSpinzor

Brawlhalla `.anm` animation file format library.

- **Project:** https://github.com/AllHailCheese/WallyAnmSpinzor
- **Copyright (c) 2024 AllHailCheese**
- **License:** MIT

### 📦 Velopack

Cross-platform installer / auto-update framework powering Odd Brawlhalla's
`Setup.exe` and in-app updates.

- **Project:** https://github.com/velopack/velopack
- **Copyright (c) 2022 Caelan Sayler**
- **License:** MIT

### 🖼 Avalonia

The cross-platform XAML UI toolkit Odd Brawlhalla is built on.

- **Project:** https://github.com/AvaloniaUI/Avalonia
- **Copyright (c) The AvaloniaUI Project**
- **License:** MIT
